# COFFEE-SALES
☕ Coffee Sales Analysis
Project Overview

This project analyzes a coffee sales dataset to uncover trends, monitor business performance, and track key performance indicators (KPIs). The analysis was carried out using Excel for data cleaning and Power BI for interactive visualizations and dashboards.

Tools Used
Microsoft Excel — Data cleaning, preparation, and exploratory analysis
Power BI — Interactive dashboards and KPI tracking

Objectives
Analyze coffee sales performance over time
Identify sales trends and seasonal patterns
Track key KPIs such as total revenue, units sold, and growth rate
Provide actionable insights to support business decisions

Key Findings
📈 Sales showed a consistent upward trend over the analyzed period
☕ Identified top-performing coffee products by revenue and volume
📊 KPIs tracked include total sales, monthly growth, and product performance
🌍 Regional and category breakdowns revealed key drivers of revenue growth
